# 🚀 איך להריץ את הפרויקט - מדריך מהיר

מסמך זה מסביר איך לקחת את הפרומפט (`PROMPT.md`) ולהפוך אותו לאפליקציה רצה אצלך במחשב.

---

## 📋 לפני שמתחילים - וידוא שיש לך הכל

פתחי טרמינל (Terminal ב-Mac, או PowerShell ב-Windows) והריצי:

```bash
node --version    # צריך להחזיר v18 או יותר
npm --version     # צריך להחזיר 9 או יותר
claude --version  # צריך להחזיר מספר גרסה של Claude Code
```

אם כל השלושה עובדים - את מוכנה 🎉

---

## 🎬 השלבים - יחד לוקחים כ-10 דקות

### שלב 1: צרי תיקייה לפרויקט

```bash
mkdir ~/recruiter-platform
cd ~/recruiter-platform
```

**הסבר:** `mkdir` יוצר תיקייה חדשה, `cd` נכנס לתוכה. הסימן `~` אומר "תיקיית הבית שלי".

### שלב 2: הפעילי את Claude Code

בתוך התיקייה החדשה, הקלידי:

```bash
claude
```

תיפתח שיחה עם Claude Code בטרמינל.

### שלב 3: הדביקי את הפרומפט

1. פתחי את הקובץ `PROMPT.md` (שהורדת לידך)
2. סמני הכל (`Cmd+A` ב-Mac / `Ctrl+A` ב-Windows)
3. העתיקי (`Cmd+C` / `Ctrl+C`)
4. עברי לטרמינל והדביקי לתוך השיחה עם Claude Code (`Cmd+V` / `Ctrl+V`)
5. לחצי Enter

**טיפ:** אם הפרומפט נראה ארוך - זה בסדר! זה הכל הוראה אחת. תני ל-Claude Code לקרוא הכל ולעבוד.

### שלב 4: חכי שיבנה

Claude Code יתחיל לעבוד. את תראי בטרמינל:
- הוא יוצר קבצים ותיקיות
- הוא מתקין חבילות (npm install)
- הוא בונה את הקוד

זה לוקח **5-10 דקות** בפעם הראשונה. את יכולה ללכת לעשות קפה ☕

**מתי לאשר?** ייתכן ש-Claude Code יבקש אישור לפקודות מסוימות (כמו "האם להריץ npm install?"). תאשרי - זה בטוח.

### שלב 5: הריצי את האפליקציה

כש-Claude Code יסיים, הוא יגיד לך משהו כמו "השלמתי את הפרויקט". עכשיו הריצי:

```bash
npm run dev
```

תראי הודעה:
```
  VITE ready in 524 ms

  ➜  Local:   http://localhost:5173/
```

פתחי בדפדפן את **http://localhost:5173/** - האפליקציה רצה! 🎊

### שלב 6: בדקי במובייל

כדי לראות איך זה נראה במובייל:
1. בכרום, לחצי `F12` (פותח את כלי המפתח)
2. לחצי על האייקון של הטלפון/טאבלט (פינה שמאלית עליונה של הכלי)
3. בחרי "iPhone 14" או "Pixel 7"
4. ראי את חוויית המובייל המדהימה שבנינו! 📱

## 🆘 בעיות נפוצות ופתרונות

### "command not found: claude"
Claude Code לא מותקן. התקיני: `npm install -g @anthropic-ai/claude-code`

### "EADDRINUSE: port 5173 already in use"
האפליקציה כבר רצה במקום אחר. סגרי את הטרמינל הקודם או הריצי `npm run dev -- --port 3000`

### האפליקציה נראית בלי עיצוב
ייתכן ש-Tailwind לא הופעל נכון. בקשי מ-Claude Code: "Tailwind לא עובד, תבדוק את ההגדרות"

### העברית מופיעה הפוך / מסודרת מוזר
בקשי מ-Claude Code: "בדוק שכל הדפים יש להם dir=rtl וטקסט מיושר לימין"

### שגיאה כשמשליחים תשובות
בקשי: "תוסיף לוגים לפונקציית calculateScores כדי שאוכל לראות מה קורה"

## 📚 מילון מונחים מהיר

- **npm** - מנהל החבילות של JavaScript (כמו App Store לקוד)
- **Vite** - הכלי שמריץ את האפליקציה במהירות בזמן פיתוח
- **React** - הספרייה לבניית ממשק משתמש
- **Tailwind** - שיטה לעצב עם CSS דרך קלאסים פשוטים
- **localStorage** - "זיכרון" של הדפדפן ששומר נתונים בין ביקורים
- **RTL** - Right To Left, ימין-לשמאל (עברית/ערבית)
- **Route** - "כתובת" בתוך האפליקציה (למשל /report/123)
- **Component** - חלק עצמאי של ממשק שאפשר להשתמש בו שוב

5. תקבלי לינק כמו `https://recruiter-platform.vercel.app` שתוכלי לשלוח לכל אחד

**יתרון נוסף:** כל שינוי שתעשי וב-`git push` יעלה אוטומטית לאתר!
