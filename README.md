# פלטפורמת אבחון אישיותי לגיוס

<div dir="rtl">

אפליקציית React בעברית RTL לשליחת שאלון אישיות BIG5 למועמדים, ניתוח התאמתם לתפקיד ספציפי, והפקת דוח חכם למגייסת.

---

## 🎯 מה זה עושה

- **המגייסת** יוצרת מועמד חדש, בוחרת לו תפקיד, ומקבלת קישור אישי לשליחה.
- **המועמד** פותח את הקישור (מותאם מובייל), עונה על 50 שאלות BIG5, ומסיים.
- **המגייסת** נכנסת לדוח שמראה: ציון התאמה כולל, פרופיל אישיות מול האידיאל, חוזקות, נקודות לתשומת לב, ושאלות מותאמות לראיון.

---

## 🧮 איך עובד הציון

1. **50 שאלות BIG5** מקובצות ל-5 ממדים: מוחצנות (E), חברתיות (A), מצפוניות (C), יציבות רגשית (S), פתיחות (O).
2. **חישוב ציון** לכל ממד בסקאלת 0-100 (כולל היפוך לשאלות הפוכות).
3. **השוואה לפרופיל אידיאלי** של התפקיד, משוקלל לפי חשיבות הממד לאותו תפקיד.
4. **התאמה כוללת** (`fit`) כאחוז + פירוט per-dimension.
5. **תובנות אוטומטיות**: חוזקות (ציון גבוה + תואם), נקודות לתשומת לב (פערים מהותיים), שאלות לראיון מותאמות.
